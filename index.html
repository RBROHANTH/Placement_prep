Shortest Path — Dijkstra + Bellman-Ford

Graph
MST, Union-Find, Advanced Graph
S1Minimum Spanning Tree: Prim's (greedy, priority queue) and Kruskal's (sort edges + Union-Find). Implement both. Understand when each is preferred. Application: cheapest network connections.

S3Hard graph problems: Word ladder (BFS on implicit graph), accounts merge (DSU), critical connections in network (Tarjan's bridges algorithm). These test deep graph understanding.
